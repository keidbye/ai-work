# 考勤统计工具 - 使用说明

## 一、工具简介

考勤统计工具是一款帮助财务人员快速统计员工考勤信息的在线工具。只需上传考勤Excel文件，系统会自动计算每位员工的加班时长、请假、迟到等数据，并可导出Excel报表。

## 二、如何打开工具

1. 双击桌面图标 `考勤统计工具.exe`
2. 等待几秒，浏览器会自动打开
3. 如浏览器未自动打开，手动访问：`http://localhost:8080`

## 三、操作步骤

### 步骤 1：上传考勤文件

- 点击虚线框区域选择文件，或直接将Excel文件拖拽到虚线框内
- 支持格式：`.xlsx`、`.xls`
- 上传成功后，虚线框会变成绿色，显示已选择的文件名

### 步骤 2：设置休息日（可选）

- 在"设置休息日"输入框中，输入需要统计为休息日的日期
- 每行输入一个日期，格式：`年/月/日`，例如：
  ```
  2026/01/01
  2026/01/02
  2026/01/03
  ```
- 休息日用于统计周末加班时长和补贴次数

### 步骤 3：设置特殊规则员工（可选）

- 在"设置特殊规则员工"输入框中，输入需要特殊计算的员工姓名
- 姓名之间用逗号或换行分隔，例如：
  ```
  张三, 李四, 王五
  ```
- 特殊规则员工将统计以下两项：
  - **19-21加班**：晚上19:00-21:00时段的加班小时数
  - **21-05加班**：晚上21:00至次日05:00时段的加班小时数
- 特殊规则员工不统计"小时数"和"周末补贴次数"

### 步骤 4：开始计算

- 点击紫色 **"开始计算"** 按钮
- 等待数据处理（通常几秒钟）
- 计算成功后会显示"计算成功！"提示，并在下方显示统计结果表格

### 步骤 5：导出Excel

- 查看统计结果确认无误后
- 点击绿色 **"📥 导出Excel"** 按钮
- 文件会自动下载到浏览器的默认下载文件夹
- 文件名格式：`考勤统计结果_时间戳.xlsx`

## 四、统计结果说明

| 列名 | 说明 |
|------|------|
| 姓名 | 员工姓名 |
| 天数 | 加班天数 |
| 小时数 | 加班小时数（特殊规则员工不显示） |
| 请假数 | 请假次数 |
| 未打卡数 | 未打卡次数 |
| 迟到天数 | 迟到天数 |
| 周末小时 | 周末加班小时数 |
| 周末补贴次数 | 周末加班补贴次数 |
| 19-21加班 | 19:00-21:00时段加班小时数（仅特殊规则员工） |
| 21-05加班 | 21:00-05:00时段加班小时数（仅特殊规则员工） |

**注意**：特殊规则员工的行会以黄色背景高亮显示，便于区分。

## 五、常见问题

**Q：上传文件后没有反应？**
A：请确保上传的是Excel文件（.xlsx 或 .xls 格式）

**Q：计算失败怎么办？**
A：请检查考勤文件格式是否正确，确保文件包含必要的考勤数据

**Q：如何关闭工具？**
A：直接关闭黑色命令行窗口即可
